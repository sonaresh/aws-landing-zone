# Workload Integration (RAM joins only)

This folder is intentionally light.
Your network team owns TGW/VPC/Firewall in a separate repo.

In workload accounts, you can automate:
- Accepting RAM shares (TGW, resolver rules, shared subnets)
- Applying baseline tags
- Validating guardrails

Implementation is environment-specific, so this folder includes templates and placeholders.
